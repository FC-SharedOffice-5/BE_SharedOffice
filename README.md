# BE_SharedOffice
Backend_SharedOfficePlatform

# **1. 기술 스택**

- **언어**: Java 17
- **개발 환경**: IntelliJ, Gradle, Spring Boot 3
- **라이브러리**: Spring Web, Spring Security, JPA, Lombok, JWT, MySQL
- **배포**: EC2, RDS(MySQL)
- **DB** : MySQL
